== Description ==
Responsive and modern theme to fully replace default bbPress theme templates and styles. Multiple color schemes added.
Option panel added for more control. Also options are accessible from customizer.

== Installation ==
= General Requirements =
* PHP: 7.0 or newer
* bbPress 2.6.2 or newer
* WordPress: 5.0 or newer

= Modern Way: =
1. Go to the WordPress Dashboard "Add New Plugin" section.
2. Search For "Helper".
3. Install, then Activate it.

= Old Way: =
1. Upload `Helper` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= PHP Notice =
* The plugin doesn't work with PHP 5.6 or older versions.

= WordPress Notice =
* The plugin should work with WordPress 4.4 to 4.9, but these versions are no longer used for testing, and they are no longer supported.
* The plugin doesn't work with WordPress 4.3 or older versions.
